# Orange Frontend

## Tech Stack:
- Docker, Node Package Manager, TypeScript/JavaScript, MUI, HTML5, CSS 

## Practice:
- Agile, Pair, XP

## Run:
- Locally
```bash
npm install
npm run build
npm run dev
```

- Run Test
```bash
npm run test 
```

- Container Docker:
```bash
docker-compose build
docker-compose up
```
